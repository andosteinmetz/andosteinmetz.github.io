Hello World!

This is the future home of my website. The current (well, not very current) version is here.

The website will attempt to meaningfully organize some combination of the following elements:

  • A blog
  • Artworks:
    • images of sculptures
    • images of ceramics
    • photographs
  • An index of my rock collection
  • Music
  • A page devoted to web design work

I am currently considering the following methods of organization:

  1. Blogform - everything shown as a scrolling blog, with categories
  2. Project based - a list of projects, each with its own page, plus a blog
  3. A combination of 1 and 2
    • Blog as homepage, with navigation to separate projects
    • Project index as homepage, with a link to the blog
  4. Something more abstract, that is neither a blog nor a portfolio, but serves both purposes

I'm considering using the following technologies:

  • github pages for hosting
  • Jekyll for blogging
  • Angular for my Spirotator
